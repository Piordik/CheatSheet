## Приоритеты операций

| Опреации           | Обозначение   |
|--------------------|---------------|
| Постфиксы          | expr++ expr-- |
| Унарные            | +ex -ex ! ~   |
| Мультипликативные  | * / %         |
| Сложение/вычитание | + -           |
| Сдвиг              | << >> >>>     |
| Отношение          | < > <= >=     |
| Сравнение          | == !=         |
| Побитовое и        | &             |
| Побитовое искл или | ^             |
| Побитовое или      | &#124;        |
| Логическое и       | &&            |
| Логическое или     |  &#124;&#124; |
| Присваивание       | = += -= итд.  |
