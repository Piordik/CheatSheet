## Регулярные выражения
#### java.util.regex

### Метасимволы для поиска совпадений границ строк или текста:

| Символ | Назначение       |
|--------|------------------|
| ^      | начало строки    |
| $      | конец строки     |
| \b     | граница слова    |
| \B     | не граница слова |
| \A     | начало ввода     |
| \Z     | конец ввода      |

### Метасимволы для поиска символьных классов:

| Символ | Назначение            |
|--------|-----------------------|
| \d     | цифра                 |
| \D     | не цифра              |
| \s     | пробел                |
| \S     | не пробел             |
| \w     | буква, цифра или _    |
| \W     | не буква, цифра или _ |
| .      | любой символ          |

### Метасимволы для обозначения количества символов – квантификаторы. 
Квантификатор всегда следует после символа или группы символов.

| Символ | Назначение    |
|--------|---------------|
| ?      | 1 или 0       |
| *      | 0 или более   |
| +      | 1 или более   |
| {x}    | x раз         |
| {x,}   | x и более раз |
| {x,y}  | [x;y]         |
